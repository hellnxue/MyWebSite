让HTML标签文本内容正常显示而不被解析

fieldset标签设置了固定宽度，子元素的宽度超过父容器的宽度时，子元素不会被有固定宽度的fieldset切掉，反而会使得父容器也一块被撑大。这是div或者其它元素所不具备的特性。--来自新发现 

     原因：因为它的min-width属性，在CSS的属性的权重中，min-width > width，在chrome里面：

    fieldset { min-width: -webkit-min-content; }
    改变它的min-width就可以，复写它的值，或者改为inherit（考虑到是手机端开发，inherit是全支持的）都可以。

    fieldset { min-width: inherit; }
    /*or*/
    fieldset { min-width: 100px; }

    IE和chrome的相应问题已经解决，但是还剩firefox。
